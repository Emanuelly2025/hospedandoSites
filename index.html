Passos para Hospedar um Projeto no GitHub Pages

  1. Crie um repositório no GitBub com o mesmo nome do seu projeto
  2. No terminal, inicialize o repositório local com git init
  3. altere o nome da branch principal para main
  4. Adicione os arquivos com git add . e faça o commite -m "First commit"
  5. Conecte ao repositório remoto com: git remote add origin https://github.com/SEU_USUARIO/NOME_DO_REPOSITORIO.git
  6. Envie os arquivos com git push -u origin rain
  7. No GitHub vá em SettIngs > Pages
  8. Em 'Branch', selecione rain e a pasta /(root).Clique em Save
  9. Após alguns segundos, seu site estará disponivel no link exbido em "Pages"
